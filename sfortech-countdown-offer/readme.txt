A lightweight shortcode-based countdown timer plugin for limited-time offers and promotions.

== Description ==

SFORTECH Countdown Offer allows you to easily display a dynamic countdown timer anywhere on your WordPress site using a simple shortcode.

Perfect for:
* Promotional campaigns
* Limited-time offers
* Product launches
* Event deadlines

The plugin is lightweight, responsive, and works with all major editors including Gutenberg, Classic Editor, and Elementor.

== Features ==

* Shortcode-based countdown timer
* Dynamic headline and sub-headline
* Custom expiry date and time
* Supports multiple countdowns per page
* Uses WordPress native jQuery
* Fully responsive
* No theme dependency

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/sfortech-countdown-offer/`
2. Activate the plugin through the "Plugins" menu in WordPress
3. Add the shortcode to any post or page

== Usage ==

Use the shortcode:

[sfortech_countdown
headline="Limited Time Offer!"
sub_headline="This exclusive offer ends soon."
expire_date="2026-02-10"
expire_time="23:59:59"]

== Shortcode Attributes ==

* headline (optional) – Main heading text
* sub_headline (optional) – Supporting text
* expire_date (required) – Expiry date (YYYY-MM-DD)
* expire_time (optional) – Expiry time (24-hour format)

== Frequently Asked Questions ==

= Does this plugin support multiple countdowns? =
Yes, you can use multiple shortcodes on the same page.

= What happens after expiry? =
The countdown stops at zero. Auto-hide functionality is planned.

= Can I customize the design? =
Yes, you can override the CSS in your theme.
